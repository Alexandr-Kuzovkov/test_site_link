Скрипт тестирования ссылок на сайте
Использование: в консоли даем команду php start.php <site_url>
В каталоге work записываются результаты работы в виде файлов:
link.fail.txt - файл со списком битых ссылок в формате: url текущей страницы: url битой ссылки
link.ok.txt -  файл со списком рабочих ссылок в формате: url текущей страницы: url рабочей ссылки
link.processed.txt - файл со списком обработанных ссылок
queue.txt - файл со списком ссылок стоящих в очереди на обработку

.testignore - файл со списком регулярных выражений. Ссылки соответсвующие хотя бы одному выражению
                из списка не включаются в очередь и не обрабатываются.
 
 При каждом запуске скрипта проверка начинается сначала
 Файлы link.fail.txt, link.ok.txt, link.processed.txt, queue.txt очищаются
